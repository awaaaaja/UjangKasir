UI/UX Specification untuk Codex

Patokan teknis desain agar Codex membangun WPF/WinForms dengan arah jelas dan tidak keluar dari konsep UjangKasir.

Prinsip Desain

Wajib
1. Kasir harus cepat.
POS tidak boleh banyak klik. Barcode input fokus otomatis.
2. Total harus besar.
Kasir langsung melihat total, bayar, dan kembalian.
3. Warna status konsisten.
Hijau aman, kuning warning, merah bahaya, biru informasi.
4. Desktop first.
Patokan layar kasir 1366x768.
